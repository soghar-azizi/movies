# 🚀 React Course Practice Repository

Welcome to the React Practice Repository 👋
This repository is designed for students following the **Ultimate React Course** to practice each section using real-world development workflow with Git and GitHub.

---

## 📚 How This Works

For every section of the course:

* You will receive a **task (assignment)** inside the Click Up folder
* You must complete the task using React
* Then submit your work via **GitHub (branch + pull request)**


---

## 🧠 Submission Rules

To keep everything organized and professional, all students must follow these rules:

---

### 🔹 1. Clone the Repository

```bash
git clone <repo-url>
```

---

### 🔹 2. Create a New Branch

Each assignment must be submitted in a separate branch.

📌 Branch naming(just firstname) format:

```bash
yourname/section-01
```

✅ Examples:

```bash
soghra/section-01
joma/section-02
```

---

### 🔹 3. Complete the Assignment

* Write clean and readable code
* Ensure your project runs without errors

---

### 🔹 4. Commit Your Changes

```bash
git add .
git commit -m "Completed Section 01"
```

📌 Use clear commit messages:

* Completed Section 02
* Added Navbar Component
* Fixed state bug

---

### 🔹 5. Push Your Branch

```bash
git push origin yourname/section-01
```

---

### 🔹 6. Create a Pull Request (PR)

* Go to the GitHub repository
* Click **Compare & Pull Request**
* Add:

  * Title: Section 01 - Your Name
  * Description: What you implemented
* Submit the PR

---

## ⚠️ Important Rules

* ❌ Do NOT push directly to the `main` branch
* ✅ Always create a new branch for each section
* ❌ Do NOT submit multiple sections in one branch
* ❌ Do NOT copy code from other students or AI
* ✅ Ask for help if you are stuck

---

## ✅ Before Submitting Checklist

Make sure:

* [ ] Project runs correctly
* [ ] Code is clean and formatted
* [ ] No unnecessary files included
* [ ] Branch name is correct
* [ ] PR title and description are clear
